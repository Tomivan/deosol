# Getting Started with Create React App
Deosol Events and Decor

# Description

Deosol events and decor is an event planning buisness that helps you organise any type of event - weddings, birthdays, proposals, bridal showers, baby showers, funerals. Info to reach them is on the contact us page. Packages they offer is on the packages page.

# Stack

. React
. Bootstrap
. FontAwesome icons

# How to run locally

git clone https://github.com/Tomivan/deosol.git

cd deosol

npm install or yarn install

npm start or yarn start
